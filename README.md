Descripción
Este es un sitio web de e-commerce estático para la marca Converse que implementa una aplicación web multi-página con funcionalidad de carrito de compras del lado del cliente. 
El sistema utiliza una arquitectura en capas con clara separación entre presentación (HTML), estilos (SCSS) y comportamiento (JavaScript)


Tecnologías Utilizadas
HTML5, SCSS (Sass), JavaScript.

El carrito se implementa completamente en el lado del cliente usando manipulación DOM. Los usuarios pueden agregar productos haciendo clic en los botones "Agregar al carrito" index.html:50-71 , lo que activa event listeners en functions.js que crean dinámicamente elementos del carrito y actualizan el contador de items
